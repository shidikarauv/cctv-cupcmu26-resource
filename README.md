# cctv-cupcmu26-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不针对任何特定网站或域名。

## 项目简介

本仓库的主要目的是为多个独立 HTML 页面提供集中存放与访问的场所。每个页面内容相互独立，可单独使用或引用。

## 目录结构

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面使用的公共资源（如样式、脚本、图片等）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件对应一个独立页面。
- `assets/` 目录存放页面共享的静态资源，按需组织。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开。
- 页面之间无依赖关系，可单独更新或删除。
- 归档内容可能不定期更新，但不会影响其他页面。

## 维护说明

- 如需新增页面，请在 `pages/` 下创建对应文件夹或文件，并确保引用资源路径正确。
- 公共资源请放入 `assets/` 目录，避免重复。
- 建议在更新后简要记录变更，便于回溯。

## 使用方式

- 直接通过 GitHub Pages 或其他静态托管服务访问 `pages/` 下的文件即可。
- 也可克隆仓库到本地，在本地浏览器中打开 HTML 文件查看。

## 注意事项

- 本仓库仅用于归档静态 HTML 页面，不涉及后端逻辑或动态内容。
- 所有页面内容由维护者自行负责，请勿存放违法或侵权信息。
